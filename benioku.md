# BenKimim - Modern Portfolyo Web Sitesi

Bu proje, modern ve şık bir portfolyo web sitesidir. React, Vite ve Tailwind CSS (v4) kullanılarak geliştirilmiştir.

## Mevcut Durum

Proje şu anda temel özelliklerini tamamlamış durumdadır:
- **Karanlık/Aydınlık Tema Desteği:** Kullanıcı tercihlerine göre geçiş yapılabilen dinamik tema sistemi.
- **Modern Animasyonlar:** Framer Motion kullanılarak akıcı geçişler ve etkileşimler.
- **Özel İmleç:** Web sitesine özgü etkileşimli custom cursor.
- **Preloader:** Sayfa açılışında marka kimliğini yansıtan yükleme ekranı.
- **Bölümler:** Hero, Hizmetler (Services), Portfolyo, Referanslar (Testimonials) ve İletişim (Support) bölümleri aktif.
- **Responsive Tasarım:** Tüm cihazlarla uyumlu yapı.
- **E-posta Entegrasyonu:** EmailJS kullanılarak çalışan iletişim formu.

## Teknoloji Yığını

- **Framework:** React 19
- **Build Tool:** Vite 6
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Email:** EmailJS

## Kurulum ve Çalıştırma

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

3. Üretim sürümü için build alın:
   ```bash
   npm run build
   ```
